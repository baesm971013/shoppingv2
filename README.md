# 🛒 쇼핑몰 관리 시스템

Streamlit을 사용한 기본 쇼핑몰 관리 시스템입니다.

## 주요 기능

1. **아이템 리스트 조회**
   - 전체 아이템 목록 표시
   - 검색 기능 (이름, 설명)
   - 카테고리 필터링
   - 다양한 정렬 옵션 (ID, 이름, 가격, 재고)

2. **아이템 추가**
   - 새로운 아이템 등록
   - 이름, 설명, 카테고리, 가격, 재고 정보 입력

3. **아이템 수정**
   - 기존 아이템 정보 수정
   - 모든 필드 수정 가능

4. **아이템 삭제**
   - 아이템 삭제 기능
   - 삭제 전 확인 절차

5. **통계 정보**
   - 전체 통계 (아이템 수, 총 재고 가치, 총 재고 수량, 평균 가격)
   - 카테고리별 통계
   - 재고 부족 아이템 알림
   - 최근 추가된 아이템 목록

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
streamlit run app.py
```

## 데이터 저장

아이템 데이터는 `data/items.json` 파일에 JSON 형식으로 저장됩니다.

## 사용 방법

1. 사이드바에서 원하는 기능을 선택합니다.
2. 아이템 조회: 검색 및 필터를 사용하여 원하는 아이템을 찾습니다.
3. 아이템 추가: 폼을 작성하여 새 아이템을 추가합니다.
4. 아이템 수정: 드롭다운에서 아이템을 선택하고 정보를 수정합니다.
5. 아이템 삭제: 드롭다운에서 아이템을 선택하고 삭제합니다.
6. 통계 정보: 전체 통계 및 카테고리별 통계를 확인합니다.

## 파일 구조

```
shoppingmall/
├── app.py              # 메인 애플리케이션
├── requirements.txt    # 필요한 패키지 목록
├── README.md          # 프로젝트 설명서
└── data/
    └── items.json     # 아이템 데이터 (자동 생성)
```
